¿Es posible volver en el tiempo?

Con Git Sí, solo debes usar el comando git reset [commit version] --hard y volverás a la version indicada (reemplazar [commit version] por la version a la que deseas volver

Me encanta GIT. He aprendido otros comandos.
git branch nombre-branch permite crear una nueva rama.
git checkout nombre-branch permite cambiar de la rama Master a la rama que especifiquemos.
git commit -am "Cambios ..." permite realizar add y commit al mismo tiempo. Solo funciona cuando hagamos hecho add y commit antes, por separado. Si creamos un archivo y queremos añadirlo con git commit -am "...", no funcionará. git merge nombre-branch -m "Cambios que serán fusionados de la rama nombre-branch al master" Este comando es como un commit, para hacer un merge de la rama nombre-branch, debo hacer checkout a mi rama master, es decir, debo cambiarme a mi rama master, y estando en ella ejecuto el comando git merge nombre-branch
